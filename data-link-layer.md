# 数据链路层

##  数据链路层的基本概念

| 名词 | 解释 |
| :--- | :--- |
| **结点** | 主机, 路由器 |
| **链路** | 网络中两结点间的`物理通道` |
| **数据链路** | 网络中两结点间的`逻辑通道` |
| **帧** | 链路层的协议**`数据单元`**, 封装网络层数据报 |

### 数据链路层的功能概述

1.  为网络层提供服务 \( `无确认无连接服务` `有确认无连接服务`  `有确认有连接服务` \)          
2.   链路管理 \( 链接的建立, 维持, 释放 \)
3.  组帧
4.  流量控制
5.  差错控制
